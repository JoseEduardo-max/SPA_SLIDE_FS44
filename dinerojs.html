O que é e o que resolve?

Dinero.js é uma biblioteca JavaScript projetada para lidar com valores monetários de forma precisa e consistente. Ela foi criada para resolver problemas comuns ao trabalhar com valores financeiros em JavaScript, especialmente aqueles relacionados à imprecisão no tratamento de números decimais. A principal vantagem do Dinero.js é garantir precisão e conformidade com os padrões financeiros ao lidar com valores monetários. Isso é importante porque os números decimais podem gerar erros de arredondamento quando usados em cálculos, o que é problemático em transações financeiras, onde a precisão é crucial.

Por que usamos DineroJS no Clube do Vinho?

A DineroJS será utilizada nessa aplicação para garantir a formatação correta dos valores dos produtos adicionados.

Como Utilizar

Na aplicação ClubedoVinho-Front podemos utilizar essa biblioteca para formatar automaticamente os preços dos produtos que foram adicionados:


function formatarPreco(valor) {
    let numero = parseFloat(valor);

    if (isNaN(numero)) {
        return 'R$ 0,00';
    }

    const preco = Dinero({ amount: Math.round(numero * 100), currency: 'BRL' });
    return preco.toFormat('$0,0.00').replace('$', '$ ');
}

O código acima é uma função chamada formatarPreco, que tem como objetivo formatar um valor monetário (passado como parâmetro) para o formato de real brasileiro (R$). A função utiliza a biblioteca Dinero.js para realizar a formatação de maneira precisa e simples. Aqui está o que o código faz:

  • Converte o valor de entrada para um número decimal.
  • Verifica se o valor é válido; caso contrário, retorna "R$ 0,00".
  • Cria um objeto Dinero usando o valor fornecido e a moeda brasileira (BRL), multiplicando o valor por 100 para trabalhar com centavos.
  • Utiliza o método toFormat do Dinero.js para formatar o número com duas casas decimais e o símbolo "$", que é substituído por "R$".
  • Retorna o valor formatado no padrão monetário.
