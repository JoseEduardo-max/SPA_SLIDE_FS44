O que é o Filterizr

Filterizr é uma biblioteca JavaScript para filtragem e animação de galerias. Ela permite organizar, filtrar e animar elementos visuais (como imagens ou cards) de forma dinâmica, suave e responsiva, sem depender de frameworks pesados.

Vantagens

  • Suporta filtros por categorias com animações fluidas em CSS3.
  • Possui reordenação automática dos elementos na grid.
  • Ideal para portfólios, catálogos e galerias de imagens.
  • Fácil de integrar com HTML, CSS e JS puro — sem dependências externas.

Desvantagens

  • Limitado a conteudo estático
  • Incompativel com acessibilidade
  • Prejudica o SEO
  • Não integra com frameworks modernos

Como instalar e aplicar

Instalação dos scripts

É necessário importar corretamente os arquivos JS e CSS do Filterizr no seu projeto.

Controles

Adicione atributos data-filter ou data-multifilter nos botões de filtro. Bem como as classes responsaveis por identificar os controles

Galeria e atributos

Atribua classes para itens filtráveis e use data-category com valores correspondentes aos filtros. Assim como as classes que identificarão os itens a ser filtrado

Função do Filterizr

Use JavaScript para instanciar o Filterizr e aplicá-lo ao container desejado.
